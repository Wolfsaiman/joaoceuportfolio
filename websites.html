Website Programming

This page is about the small websites that I made so far. I started doing website projects for school and I really started to like it. That helped me making better website designs and functions for the school projects.

There are only 2 websites in this page, and they're also small. However, I'm going to make a better description about each one of them.

FNAF Test Website

This is, as I said, a school project that I made and also the first website test that I made. This website page is about Five Nights at Freddy's (AKA FNAF), an horror game that I really love and decided to use as the project theme. The website was supposed to be a blog website, where you could check all the news about the most recent FNAF game, "FNAF: Security Breach". I only needed to make one page, so I decided to do the Home page, with the featured posts made by the community, a little description about the website, the game developer's social media, etc...

Saimart Website

This is another school project I made but for a different subject. I needed to make a "company" with my logo, merch, website, etc... And my favorite thing to do was, of course, programming the website. I made a full website about my company called "Saimart" with a home page, an "about" page, a portfolio page the store page and the contacts page.

Home Page

This is Saimart's home page. This page shows you every page the website has with a short explanation about each one of them. It also gives you access to each one of those pages at the end of it.

About Page

This page tells you all you know about Saimart. It tells you how they started, how was the logo made and who is the CEO.

Portfolio Page

This page shows Saimart's Portfolio. Basically I put everything I have in this portfolio website but one example for each kind of project.

Store Page

I wanted to add some humor to this page, so for the Store Page I decided to make a little message saying that "Due to (this website is only a school project) this page is currently unavailable". Just a little joke since my company wouldn't have a real store in a school project.

Contacts Page

This is a simple Contacts Page with nothing more than all the 3 contacts that Saimart has, such as e-mail, phone number and physic headquarter.
